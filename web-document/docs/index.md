# rhodash

::: warning
rhodash is now on beta.
:::

## Motivation

Lodash is huge. It is not suitable for frontend development.

## Getting Started

```bash
npm install rhodash
# or yarn add rhodash
```

## Functions

### pick(object, string[])

**Example**

```ts
pick({ a: 1, b: 2, c: 3 }, ['a', 'b']) // { a: 1, b: 2 }
```
